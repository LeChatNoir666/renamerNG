Regular expressions

One of the most powerful features of RenamerNG is the possibility to execute regular expressions. If you are new to regular expressions you can have a look at http://www.regular-expressions.info to learn more.


Command Line Interface (CLI)

RenamerNG now as an experimental CLI that enables the user to enter commands (operations or macros) as text and execute them.
Syntax:

operation or macro name[|parameter1][|parameter2][...]
The syntax checking is not that advanced and you can easily cause a command to fail by spelling wrong, but If you like to tinker with it, it's available. If many users like this feature it will be further developed. To specify a | in for example a replace operation use \| other escapes are \n for new line and \" for double quote.


Parameters during startup

RenamerNG can be started with three different sets of parameters if none or one parameter is given RenamerNG will start and scan the dir if specified.
Syntax:

RenamerNG [directory]

directory - Should be a valid directory used for simple shell integration


OnScan macro

After each time a directory is scanned RenamerNG will look for a macro named "OnScan" if this macro exists it will be automatically executed. This allows you to record any operations in this macro that you want to be executed when you scan a directory.
